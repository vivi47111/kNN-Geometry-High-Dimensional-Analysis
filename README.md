# Geometric Analysis of k-NN in High-Dimensional Spaces

## 1. Project Overview
This project provides a mathematical exploration of the **Curse of Dimensionality**. By analyzing the geometric properties of the **k-Nearest Neighbors (k-NN)** algorithm within a unit hypercube, I demonstrate how the concept of "locality" degrades as the number of dimensions increases.

## 2. Key Research Highlights
* **Radius Expansion Analysis**: Quantified how the neighborhood radius must increase to capture a fixed fraction of data as dimensions grow.
* **Convergence Velocity**: Analyzed the **Error Rate Decay Velocity** to compare the efficiency of **LDA** vs. **Logistic Regression** across various sample sizes.
* **Risk Decomposition**: Evaluated the Bias-Variance Tradeoff to identify the optimal complexity for high-dimensional predictors.

## 3. Technical Implementation
* **Geometric Modeling**: Theoretical derivation of volume concentration in high-dimensional hypercubes.
* **Comparative Benchmarking**: Performance analysis of generative vs. discriminative models under high-dimensional constraints.
